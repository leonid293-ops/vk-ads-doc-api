## /api/v2/agency/managers//clients/.json
## /api/v2/agency/managers//clients/.json

Ресурс позволяет управлять учетными записями клиентов, находящихся в ведении менеджера агентства.

Ресурс доступен только пользователям-агентствам.

Используется объект: [AgencyManagerClient](../object/AgencyManagerClient)

### POST

Запрос редактирует данные и права доступ клиента, находящегося в ведении менеджера.

Пример запроса на редактирование клиента:

```HTTP

  POST /api/v2/agency/managers/123/clients/17668.json
  {
    "access_type":"full_access",
  }
```

где:
123 - это ID менеджера в ведении агентства;

17668 - это ID клиента.

Возвращаемые коды статуса:

204 - Данные клиента были успешно изменены.

400 - Ошибки валидации (код "validation_failed").

404 - Клиент не найден.

Примеры ошибок:

```HTTP

  {"error": {"code": "validation_failed", "fields": {  }}}
 - Ошибка в значениях передаваемых данных.
```

### DELETE

Запрос выводит клиента (удаляет) из ведения менеджера. Операция не приводит к полному удалению учетной записи клиента или выведению клиента из ведома агентства.

Пример запроса:

```HTTP

  DELETE /api/v2/agency/managers/123/clients/17668.json
```

где:
123 - это ID менеджера в ведении агентства;

17668 - это ID клиента.

Возвращаемые коды статуса:

204 - Клиент, находящиеся в ведении менеджера, был успешно удален.

404 - В следующих случаях:
- Клиент не находится в ведении указанного менеджера.
- Менеджер не находится в ведении агентства.
