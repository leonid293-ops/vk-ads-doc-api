## /api/v2/billing/transactions//.json

Ресурс выполняет перевод средств между агентством и его клиентом

Используется объект: [Transaction](../object/Transaction)

### POST

В теле запроса указывается структура вида:

```HTTP

   {"amount": "10000.00"}
```

где  - сумма перевода

 - направление перевода, может принимать 2 значения:
to - перевод средств от агентства клиенту; 
from - перевод средств от клиента агентству

Пример запроса перевода средств клиенту:

```HTTP

   POST /api/v2/billing/transactions/to/1234567.json
```

где user_id клиента - 1234567

Пример ответа:

```HTTP

   {
       "amount": "10000.00",
       "created_at": "2018-11-21 13:51:28",
       "client_balance": "23000.00",
       "agency_balance": "150000.00",
       "client_username": "name@agency_client",
   }
```

Ответ содержит данные после проведения транзакции.

На счетах аккаунтов должно быть достаточно средств для переводов. Коды ответов:

200 - транзакция успешно проведена

400 - ошибка перевода (код "transfer_error").
Проверьте балансы и повторите запрос позже.

400 - ошибка валидации суммы остатка у клиента (код "failed_amount"). 
Сумма превышает доступный лимит, детали в "message"

400 - клиент не активен (код "blocked_user")

404 - клиент на найден (код "not_found")

Пример ответа с ошибкой:

```HTTP

   {
       "error": {
           "message": "Can't transfer amount 10000.00",
           "code": "transfer_error",
       }
   }
```
